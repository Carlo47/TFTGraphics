# TFTGraphics

Shows a menu on a TFT display and allows the selection of 
everal functions by means of a pushbutton.
The graphical examples are:
- Digital Clock
- Analog Clock
- Lines
- Rectangles
- Rounded Rectangles
- Circles
- Triangles
- Sierpinsky Triangle
- Mandelbrot Fractal
- Barnsley Fern
- Random Walk
- Random Points
- HSV Color Circle
- RGB Color Space
- HSV Color Space
- Rotated Text
- Text Sizes
- Fonts

The folder TFTGraphics contains the full PlatformIO project.
